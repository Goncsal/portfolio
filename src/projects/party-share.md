# PartyShare

Project Summary: PartyShare is a comprehensive rental platform for event equipment, featuring a robust CI/CD pipeline, automated testing, and high-quality software engineering practices.

## Overview

PartyShare enables users to rent and list items such as generators, lighting equipment, sound systems, decorations, and other party and event equipment. This project was developed with a strong focus on software quality and modern DevOps practices.

- **Quality Assurance**: Full test coverage and quality assurance practices.

## Technical Implementation

- **Backend**: Built with Java (primary language).
- **Deployment**: Dockerized application running on deti-tqs-08.ua.pt.
- **API Documentation**: Available via Swagger.
- **Architecture**: Includes automated testing, continuous integration/deployment pipelines, and static code analysis.

## Key Features

- **Item Listing**: Users can list their party equipment for rent.
- **Rental Management**: Seamless process for renting items.
- **Automated Testing**: Comprehensive test suite ensuring reliability.
- **Scalable Infrastructure**: Dockerized for easy deployment and scaling.
